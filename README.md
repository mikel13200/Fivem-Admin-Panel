### Requirements
* Web Server w/ PHP
* VPS (To host the API)
* MySQL Database
* Steam API Key (https://steamcommunity.com/dev/apikey)


### Installation
* Install the web files in the httdocs folder of xamp or in the www folder of wamp64, then open the api and replace the api url in the .html files.
